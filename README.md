# Customer Churn & Retention Analysis

## Problem Statement
Analyze customer churn and identify key factors affecting retention in a subscription-based business.

## Objectives
- Understand churn patterns
- Identify high-risk customer segments
- Analyze customer lifetime trends
- Suggest strategies to improve retention

## Tools Used
- Excel / Power BI
- (Optional: Python)

## Key Insights
- High churn observed in monthly subscription users
- Customers with shorter tenure are more likely to leave
- Lack of engagement is a major churn driver
- Long-term users show higher retention

## Recommendations
- Offer discounts for long-term plans
- Improve onboarding experience
- Increase engagement via notifications/emails
- Provide loyalty rewards

## Files Included
- Dashboard (.pbix / Excel)
- Presentation (.pptx)
- Dataset

## Outcome
This project demonstrates real-world business analysis and data-driven decision making.
